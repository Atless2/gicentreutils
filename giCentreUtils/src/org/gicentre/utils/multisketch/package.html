Multiple and embedded sketch utilities for use with Processing

This package includes classes for creating multiple sketches and for embedding sketches in other Java components. Includes classes for creating PowerPoint-like presentations with embedded sketches.

To check you have the latest version of the gicentreUtils package, display the results of Version.getText() or Version.getVersion(). It should match that shown in this API documentation.
